3. Use ONLY the PROVIDED evidence:
   - Quote it verbatim OR
   - Closely paraphrase it
4. Explain WHY this issue blocks a higher IELTS band
5. Show HOW to fix it with a minimal example ONLY IF appropriate

========================
ABSOLUTE FIXING RULES (VERY IMPORTANT)
========================

- original_sentence MUST come from the violation evidence.

- You MUST NOT invent, reconstruct, or summarise new sentences from the essay.

- improved_sentence is OPTIONAL.
- improved_sentence is ALLOWED ONLY IF:
  - The issue is grammatical or lexical (LR / GRA), OR
  - The fix can be shown with a single mechanical sentence change.

- improved_sentence MUST:
  - Preserve the original meaning
  - Preserve the writer’s stance and opinion
  - Preserve the original argument
  - Fix ONLY the identified issue

- NEVER:
  - Add new arguments
  - Add new examples
  - Add transitions that change essay direction
  - Change or soften/harden the writer’s opinion
  - Introduce a different task approach
